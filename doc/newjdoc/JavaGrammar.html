EPP が受理する Java 文法

このドキュメントは、 EPP が受理する Java 文法を、拡張 BNF 記法で 書いたものです。

plug-in は、このドキュメント中の非終端記号に新たな選択肢を追加して、 文法を拡張することができます。

実装上、非終端記号には２種類あり、拡張方法が違います。

  • 非終端記号のうしろに # がついていないものは、 defineNonTerminal マクロを使って定義された非終端記号です。 xxxTop, xxxLeft, xxxRight というメソッドを拡張することで、 前置演算子、左再帰ルール、右結合演算子などの選択肢を追加できます。 メソッド xxx そのものは、拡張してはいけません。
  • 非終端記号のうしろに # がついているものは、defineNonTerminal を 使わない、普通のメソッドとして定義されているものです。 この種類の非終端記号には、 xxxTop, xxxLeft, xxxRight は使えません。 メソッド xxx を拡張して選択肢を追加します。
（ xxx は、非終端記号の先頭の文字を小文字にしたものです。） 
メタ記号について：
・空 (<e>)
・直前の記号の0回以上の繰り返し (<*>)
・直前の記号二つの0回以上の繰り返し (<**>)
・直前の記号二つの1回以上の繰り返し (<..>)
・直前の記号二つ(A Bとする)の、A B <**> Aの形の繰り返し (<...>)
